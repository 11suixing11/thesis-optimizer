# 优化任务契约

## 概述

本文档定义了优化任务的预提交计划，确保每次优化都有明确的目标、范围和验证标准。

## 契约模板

```markdown
# 优化任务契约

## 任务信息
- **任务ID**：[自动生成]
- **章节**：[章节号]-[章节名]
- **优化类型**：[降AI / 降重 / 润色 / 格式]
- **创建日期**：[日期]
- **执行Agent**：[agent名称]

## 优化目标
- **目标指标**：[AI检测率<20% / 查重率<10% / 格式规范]
- **当前值**：[当前指标值]
- **目标值**：[目标指标值]

## 优化范围
- **目标段落**：[段落号列表]
- **涉及行数**：[行号范围]
- **保护内容**：[术语/公式/引用列表]

## 执行计划
1. [步骤1]
2. [步骤2]
3. [步骤3]

## 验证标准
- [ ] 语义一致性验证
- [ ] 术语保留验证
- [ ] 指标达标验证
- [ ] 格式规范验证

## 预期输出
- 修订日志：[文件路径]
- 优化后文本：[文件路径]
- 评估报告：[文件路径]

## 风险评估
- **可能风险**：[风险描述]
- **应对措施**：[应对方案]
```

## 使用说明

1. 每次优化前，Agent 必须生成此契约
2. 用户确认后方可执行优化
3. 优化完成后，对照契约验证结果
4. 契约随修订日志一起保存
