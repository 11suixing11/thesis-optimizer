# AI 高频词汇风险参考

本文档列出中文学术论文中常见的AI高频词汇和短语，按风险等级分类。

## 高风险（出现即标记）

连接词：首先/其次/再次/最后 → 删除或改为自然过渡；第一/第二/第三 → 改为具体论述；不仅...而且... → 简化为直接陈述

总结类：综上所述、总而言之、概括来说、综合来看 → 删除或改为具体总结

强调类：值得注意的是、不难发现、显而易见、毋庸置疑、众所周知 → 删除或直接陈述

模糊量化：较高的、较好的、显著的、大量的、近年来 → 给出具体数值/指标/年份

## 中风险（连续出现标记）

论述类：本文提出 → 改为主动语态或具体描述；本文认为 → 改为具体分析；实验结果表明 → 改为具体数据呈现

转折类：然而、但是 → 适度使用，不超过3次/千字

## 低风险（密集使用标记）

逻辑类：因此（≤5次/千字）、所以（≤3次/千字）、由此（≤3次/千字）

学术类：研究表明（≤3次/千字）、文献指出（≤3次/千字）

## 使用说明

1. `analyzer_agent` 扫描时对照此表标记风险段落
2. `ai_reduction_agent` 优化时参照替换建议修改
3. 替换建议仅为参考，需根据上下文调整
4. 核心专业术语不在本表范围内，永远保护原文
