# AI 高频词汇风险参考

## 概述

本文档列出中文学术论文中常见的AI高频词汇和短语，按风险等级分类。

## 高风险词汇（出现即标记）

### 连接词类
| 词汇 | 替换建议 |
|------|---------|
| 首先/其次/再次/最后 | 直接删除或改为自然过渡 |
| 第一/第二/第三 | 改为具体论述 |
| 不仅...而且... | 简化为直接陈述 |
| 一方面...另一方面... | 拆分为独立句子 |

### 总结类
| 词汇 | 替换建议 |
|------|---------|
| 综上所述 | 删除或改为具体总结 |
| 总而言之 | 删除 |
| 概括来说 | 删除 |
| 综合来看 | 删除 |

### 强调类
| 词汇 | 替换建议 |
|------|---------|
| 值得注意的是 | 删除或直接陈述 |
| 不难发现 | 删除 |
| 显而易见 | 删除或改为具体证据 |
| 毋庸置疑 | 删除 |
| 众所周知 | 删除或改为引用 |

### 模糊量化类
| 词汇 | 替换建议 |
|------|---------|
| 较高的 | 给出具体数值 |
| 较好的 | 给出具体指标 |
| 显著的 | 给出统计显著性 |
| 大量的 | 给出具体数量 |
| 近年来 | 给出具体年份 |

## 中风险词汇（连续出现标记）

### 论述类
| 词汇 | 替换建议 |
|------|---------|
| 本文提出 | 改为主动语态或具体描述 |
| 本文认为 | 改为具体分析 |
| 实验结果表明 | 改为具体数据呈现 |
| 分析表明 | 改为具体分析过程 |

### 转折类
| 词汇 | 替换建议 |
|------|---------|
| 然而 | 适度使用，不超过3次/千字 |
| 但是 | 适度使用 |
| 尽管如此 | 改为具体让步 |

## 低风险词汇（密集使用标记）

### 逻辑类
| 词汇 | 使用建议 |
|------|---------|
| 因此 | 不超过5次/千字 |
| 所以 | 不超过3次/千字 |
| 由此 | 不超过3次/千字 |
| 基于此 | 不超过3次/千字 |

### 学术类
| 词汇 | 使用建议 |
|------|---------|
| 研究表明 | 不超过3次/千字 |
| 文献指出 | 不超过3次/千字 |
| 理论上 | 不超过2次/千字 |

## 使用说明

1. `analyzer_agent` 扫描论文时，对照此表标记风险段落
2. `ai_reduction_agent` 优化时，参照替换建议进行修改
3. 替换建议仅为参考，需根据上下文灵活调整
4. 核心专业术语不在本表范围内，永远保护原文
