# 句式节奏与统计指标

本文档定义评估文本"人类感"的统计指标，指导 `ai_reduction_agent` 的优化方向。

## 核心指标

### 1. 句长标准差

定义：段落中各句长度（字符数）的标准差。目标值：> 8

| 标准差 | 风险 | 说明 |
|--------|------|------|
| < 3 | 高 | 句式高度一致，明显AI特征 |
| 3-5 | 中 | 句式较规整 |
| 5-8 | 低 | 基本正常 |
| > 8 | 安全 | 句式多样，人类特征明显 |

优化：短句（<15字）适当扩展；长句（>40字）拆分；等长句故意制造长短差异。

### 2. 词汇丰富度（TTR）

定义：不同词汇数 / 总词汇数。目标值：> 0.6

| TTR | 风险 | 说明 |
|-----|------|------|
| < 0.4 | 高 | 词汇重复严重 |
| 0.4-0.5 | 中 | 词汇较单一 |
| 0.5-0.6 | 低 | 基本正常 |
| > 0.6 | 安全 | 词汇丰富 |

优化：替换高频重复词；使用术语的多种表述；增加形容词和副词变化。

### 3. 困惑度（Perplexity）

衡量文本可预测性。AI文本困惑度较低（太规整），人类写作困惑度较高（有随机性）。

优化：引入适度"意外"元素；使用非常规句式；保留人类写作的不完美性。

### 4. 突发性（Burstiness）

衡量信息密度变化。人类写作信息密度有起伏，AI写作信息密度均匀。

优化：重要论点处增加信息密度；过渡处适当降低；模拟人类写作的"起伏"感。

## 应用

优化前：计算目标段落的句长标准差和TTR，标记低于阈值的段落。

优化后：重新计算指标，确认达标。未达标则继续优化。

注意：这些指标是参考而非绝对标准，学术论文本身有一定规整性要求，需结合学科特点灵活应用。
