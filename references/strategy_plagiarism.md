# 降查重率策略

## 概述

本文档为 `plagiarism_agent` 提供详细的降查重策略。

## 核心原则

### ⚠️ IRON RULE

- 语义必须与原文完全一致
- 专业术语必须保留
- 不添加原文没有的信息
- 不删除核心论点

## 策略一：深度语义改写

### 1.1 同义替换

**范围**：非核心词汇（动词、形容词、副词）

**示例**：
| 原文 | 替换 |
|------|------|
| 提出 | 给出/设计/构建 |
| 有效 | 切实/可行/奏效 |
| 显著 | 明显/突出/可观 |
| 采用 | 使用/运用/利用 |
| 表明 | 显示/揭示/指出 |

**注意**：核心术语（如"卷积神经网络"）不可替换。

### 1.2 句式重组

**主被动互换**：
- 原文：「实验验证了该方法的有效性」
- 改写：「该方法的有效性经由实验得到了验证」

**因果倒置**：
- 原文：「由于X导致了Y」
- 改写：「Y的出现可归因于X」

**拆分合并**：
- 长句拆短句：将一个复杂句拆分为2-3个简单句
- 短句合长句：将多个简单句合并为一个复合句

### 1.3 表述角度转换

- 原文：「该方法的准确率达到了95%」
- 改写：「在准确率指标上，该方法取得了95%的成绩」

- 原文：「本文设计了一种新的算法」
- 改写：「一种新的算法在本文中被设计出来」

## 策略二：引用规范化

### 2.1 直接引用 → 间接转述

**原文**：
> 张三(2020)指出："深度学习在图像识别领域取得了显著进展"

**改写**：
> 张三(2020)的研究表明，深度学习技术已在图像识别方面实现了重要突破

### 2.2 整合多篇文献

**原文**：
> 张三(2020)研究了A。李四(2021)研究了B。王五(2022)研究了C。

**改写**：
> 近年来，多位学者从不同角度对该问题进行了深入研究。张三(2020)聚焦于A方面，李四(2021)则从B角度展开探讨，而王五(2022)进一步将研究拓展至C领域。

### 2.3 添加个人分析

在引用后加入自己的分析和评价：
> 张三(2020)的研究表明...。该观点为本研究提供了理论支撑，但在XX方面仍有待深入探讨。

## 策略三：专业术语处理

| 处理方式 | 适用场景 | 示例 |
|---------|---------|------|
| 术语保留 | 核心概念 | "卷积神经网络"不改 |
| 扩展描述 | 术语首次出现 | "CNN（卷积神经网络，一种深度学习架构）" |
| 上下文变换 | 术语重复出现 | 第二次用"该网络架构"替代 |

## 策略四：教科书表述改写

**特征**：与教科书或常见文献高度相似的标准表述。

**改写方法**：
1. 改变表述顺序
2. 增加限定条件
3. 添加具体例子
4. 转换为问题导向的表述

## 执行流程

1. 识别查重高风险段落
2. 判断重复类型（直接引用/教科书表述/方法论雷同）
3. 选择对应策略
4. 执行改写
5. 验证语义一致性
6. 记录修订日志
