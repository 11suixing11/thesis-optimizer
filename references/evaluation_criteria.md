# 评估标准与方法

本文档定义 `evaluator_agent` 使用的评估标准和方法。

## 评估维度

### 1. AI检测风险（权重 30%）

| 检查项 | 评分标准 | 分值 |
|--------|---------|------|
| 句式多样性 | 句长标准差 > 8 得满分 | 0-20 |
| 词汇丰富度 | TTR > 0.6 得满分 | 0-20 |
| AI高频词 | 每发现1个扣5分 | 0-20 |
| 规整化结构 | 每发现1处扣5分 | 0-20 |
| 语气自然度 | 整体评估 | 0-20 |

### 2. 查重风险（权重 30%）

| 检查项 | 评分标准 | 分值 |
|--------|---------|------|
| 语义改写深度 | 是否真正改写 | 0-25 |
| 引用规范性 | 引用格式正确 | 0-25 |
| 术语处理 | 术语保留，描述改写 | 0-25 |
| 表述原创性 | 无教科书式雷同 | 0-25 |

### 3. 学术质量（权重 25%）

| 检查项 | 评分标准 | 分值 |
|--------|---------|------|
| 表达精准度 | 无模糊表述 | 0-20 |
| 术语一致性 | 全文统一 | 0-20 |
| 逻辑连贯性 | 段落间逻辑清晰 | 0-20 |
| 时态规范 | 时态正确 | 0-20 |
| 可读性 | 长短句错落 | 0-20 |

### 4. 格式规范（权重 15%）

| 检查项 | 评分标准 | 分值 |
|--------|---------|------|
| 文档结构 | 章节完整、编号正确 | 0-25 |
| 排版规范 | 格式正确 | 0-25 |
| 图表规范 | 编号标题正确 | 0-25 |
| 引用格式 | 文内引用与参考文献一致 | 0-25 |

## 评分体系

| 总分 | 等级 | 判定 |
|------|------|------|
| ≥ 85 | A | 达标，可提交 |
| 70-84 | B | 基本达标，建议微调 |
| 55-69 | C | 未达标，需返工 |
| < 55 | D | 严重不达标，需大幅返工 |

## 评估方法

### 定量评估

1. 运行 `scripts/text_stats.py` 计算句长标准差和TTR
2. 运行 `scripts/ai_vocab_checker.py` 扫描AI高频词
3. 统计各维度扣分项

### 定性评估

1. 逐段阅读，评估语气自然度
2. 检查逻辑连贯性
3. 评估学术规范性

### 评估报告

生成 `evaluation_report.md`，包含：总分和等级、各维度得分、问题清单、优化建议、判定结果。
