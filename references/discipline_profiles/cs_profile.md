# 计算机科学学科画像

## 概述

本画像定义了计算机科学领域学术论文的写作规范和特点。

## 引用格式

- **首选**：IEEE
- **备选**：ACM
- **文内引用**：[1] 或 (Author, Year)
- **参考文献排列**：按引用顺序排列

## 术语特点

### 核心术语（不可修改）
- 深度学习、机器学习、神经网络
- 卷积神经网络（CNN）、循环神经网络（RNN）、Transformer
- 梯度下降、反向传播、损失函数
- 准确率、召回率、F1值、精确率
- 过拟合、欠拟合、正则化
- 数据集、训练集、测试集、验证集

### 常见缩写（首次出现需给出全称）
- AI - Artificial Intelligence
- ML - Machine Learning
- DL - Deep Learning
- NLP - Natural Language Processing
- CV - Computer Vision
- GPU - Graphics Processing Unit
- API - Application Programming Interface

## 句式特点

- 方法描述常用主动语态
- 实验结果常用数据支撑
- 图表使用频繁
- 代码片段可作为补充

## 常见结构

1. 摘要
2. 引言（问题背景、研究动机、贡献总结）
3. 相关工作
4. 方法（模型架构、训练策略、损失函数）
5. 实验（数据集、评估指标、实验结果、消融实验）
6. 讨论
7. 结论

## 格式规范

- 正文：宋体/Times New Roman 小四号
- 标题：黑体/Times New Roman 各级标题对应字号
- 行距：1.5倍行距
- 代码：等宽字体，五号

## 特殊注意事项

- 公式编号右对齐
- 图表编号连续
- 算伪代码需规范格式
- 引用格式需全文一致
